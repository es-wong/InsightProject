Font Palettes on Demand

This project was completed in three weeks as part of the Insight Data Science Fellowship.

Fontastic Four is a tool designed to help burgeoning graphic designers select stylistically consistent font palattes.

Recommendations for font pairings are provided based on a blend of similar and contrasting features of fonts in the Google Font library.


For more information, see the slides.

Good font pairings blend similarity and contrast.

Bad font pairings are either too similar or vastly different.
